CaesarPlus

Its more secure than caesar cypher

Fast, Simple and Easy to use

CaesarPlus is written in python and can ba downloaded from the Python package index

pip install caesarplus

The more data the more secure

CaesarPlus creates a unique key to decrypt your data so the bigger it is the more secure it is. A key can range from 1character to 1,000,000 characters

Usage

Start by installing python

Unix based

To install python use the apt command to install it to your system

sudo apt install python3

Windows based

Download the installer from the python website


Downloads pageDownload python 3.10 amd64

Now install CaesarPlus

To instal CaesarPlus use the pip command

pip install caesarplus

Now encode your first encryption using CaesarPlus

Open the python terminal and run

Python

import caesarplus as cp
cp.encode("Example")

Output

([4, 4, 1, 7, 5, 0, 6], 'I|btulk')

Now decode your first encryption using CaesarPlus

Open the python terminal and run

Python

import caesarplus as cp
cp.decode([4, 4, 1, 7, 5, 0, 6],'I|btulk')

Output

'Example')
