What is GPG?

GPG is a open source tool that allows you to encrypt and sign data and communication. It is a tool that is used by many people to secure their data and communication.

What do I use it for?

Sometimes I sign messages that I send to people, this is to prove that the message was sent by me and not someone else.

Test it works

If you want to test that the key works, you can use the following command:

                    
gpg --import pub.key

Then save the following text to a file called test.txt:

                    
-----BEGIN PGP SIGNED MESSAGE-----
Hash: SHA512

This is a demo file
